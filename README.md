# メモアプリケーション

## 概要
Laravelでメモアプリを作成しました。

## 目的
* フレームワークの恩恵を理解すること
* 静的ファイルを設置できるようになること
* ログイン周りの機能を少しカスタマイズできるようになること
* セッションを使えるようになること

## 機能要件
* ログインしてメモを投稿する
* ミドルウェアを使って認可処理をする
* セッションを利用して選択しているメモを判定する
* メモのCRUD操作ができる

## データベース設計
### usersテーブル

|カラム|型|
|:-:|:-:|
|id|数値|
|name|文字列|
|email|文字列|
|email_varified_at|文字列|
|password|文字列|
|created_at|日付|
|updated_at|日付|

### memosテーブル

|カラム|型|
|:-:|:-:|
